Command-line parameters for helper.exe
/S /INSTALL – silent installation

/PRO – installation of Professional Plus

/WORD – installation of WORD
/EXCEL – installation of EXCEL
/POWERPOINT – installation of POWERPOINT
/ACCESS – installation of ACCESS (not available in the Standard version)
/OUTLOOK – installation of OUTLOOK
/ONENOTE – installation of ONENOTE
/XTEAMS – installation of MICROSOFT TEAMS
/PUBLISHER – installation of PUBLISHER

/XSTD – installation of Standard

/WORD – installation of WORD
/EXCEL – installation of EXCEL
/POWERPOINT – installation of POWERPOINT
/OUTLOOK – installation of OUTLOOK
/ONENOTE – installation of ONENOTE
/XTEAMS – installation of MICROSOFT TEAMS
/PUBLISHER – installation of PUBLISHER

/VISIOPRO – installation of Visio Pro
/VISIOSTD – installation of Visio Standard

/PROJECTPRO – installation of Project Pro
/PROJECTSTD – Project Standard installation

/LANG=ID – localization selection
where ID is the language identifier, for example RU-RU, EN-US, UK-UA
The parameter can be specified multiple times, the first language will be the main one

/UPD – allow installation of Office updates (can be changed later in the settings)
/KMS – offline activation using AAct and creating a reactivation task every 10 days
/KMS /AACT – offline activation using AAct without creating a reactivation task
/KMS /OHOOK – offline activation using MAS (Ohook method – permanent activation)
/KMS /ONLINEKMS =URL – online activation using a remote KMS server (kms.loli.best by default)
/KMS /MAS – online activation using MAS and a remote KMS server
/KMS /MONDO – online activation and installation of a Mondo license (activates some Office 365 features)
/RETTOVL – convert Retail to VL (for example, to use MAK keys)
/MAK =key - install MAK key
/OFFICETAB - install Office Tab
/UBITRU - install Russian version of UBit Menu
/UBITEN - install English version of UBit Menu
